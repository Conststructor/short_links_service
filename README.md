# short_links_service - приложение для создания коротких ссылок.

## Основные комманды
__0 - завершить программу__

***1 - создать короткую ссылку.***   
    Можно создать несколько ссылок. При создании необходимо указать количество доступных
    переходов по ссылке и время её действия в формате ч:м:с. При достижении одного из заданных ограничений ссылка 
    удаляется и её дальнейшее использование становится невозможным

***2 - перейти по короткой ссылке.***  
Нужно ввести результат создания короткой ссылки. 
    При выборе данного пункта для удобства сначала будут отображены все доступные короткие ссылки
    одну из которых можжно скопировать, вставить в терминал для перехода

***3 - изменение параметров короткой ссылки.***  
Подменю в котором можно изменить количество преходов по ссылке, 
    время действия ссылки либо удалить какую либо короткую ссылку из доступных)
